# BrainFuck
BrainFuck Interpreter written on C#.

# How to use

## Arguments in cmd:

1. brainfuck [file name]

Put on [file name] your path to BrainFuck code and press enter and your code let run!

2. brainfuck --version

This command print Interpreter version.
